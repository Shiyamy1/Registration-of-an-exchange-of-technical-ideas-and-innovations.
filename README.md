# SmartParts v3 & MiniShop — Биржа технологических идей

Живая демонстрация: консольные версии на Python  
Приветствуются любые предложения и идеи по улучшению!

---

## Описание

Это интерактивная платформа для обмена инженерными и технологическими идеями и демонстрации их применения.  

Проект состоит из двух частей:

1. SmartParts v3 — Биржа технологических решений. Позволяет:
   - искать идеи по ключевым словам;
   - добавлять новые детали и изобретения;
   - голосовать за понравившиеся решения;
   - получать случайные идеи для вдохновения;
   - просматривать рейтинг всех идей.

2. MiniShop — Мини-магазин идей и технологических продуктов. Позволяет:
   - регистрировать пользователей;
   - просматривать каталог продуктов;
   - оформлять заказы с расчетом даты доставки;
   - оставлять отзывы на полученные товары;
   - отслеживать историю заказов и рейтинг продуктов.

Проект вдохновлён идеей создания цифровой биржи технологических идей, где каждый пользователь может внести свой вклад и оценить идеи других.

---

## Как использовать

### SmartParts v3

Поиск и вдохновение
- Введите ключевые слова для поиска деталей/идей.


- Получите список результатов с рейтингом и тегами.  


- Случайная генерация идей помогает найти вдохновение.

Добавление и голосование
- Добавьте собственное изобретение с описанием и тегами.  

- Голосуйте за идеи других пользователей (+/- голоса). 

- Популярные идеи отображаются в топ-листе.

### MiniShop

Регистрация и покупки
- Зарегистрируйтесь или войдите в систему.  

- Выберите товар из каталога. 

- Оформите заказ с расчётом даты доставки.  

- После получения заказа оставьте отзыв.

 

Просмотр истории
- Проверяйте статус своих заказов и отзывы.  
- Сортировка по дате доставки и рейтингу.

## О проекте

Поиск: реализован через ключевые слова и подсчёт совпадений.  
Голосование: каждая идея имеет рейтинг, который можно увеличивать или уменьшать.  
Случайные идеи: случайная выборка деталей для вдохновения.  
MiniShop: демонстрация механизма «монетизации» идей и получения обратной связи от пользователей.
##  Технологии в проекте

 - Язык: Python 3.8+  
- Модули: random, datetime (стандартная библиотека)  
- Интерфейс: консольный  
- Формат хранения данных: в памяти (возможна интеграция с JSON / БД)  
- Возможное расширение: веб-интерфейс на Vue.js / React
